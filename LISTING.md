# Расшифровка используемых в коде проги конструкций
Используемые типы данных:
```java
public class Main {
    boolean isChecked = true; // булева перменная (true/false)
    int number = 200; // целочисленная переменная (например, 200)
    int[] arrayNumbers = new int[]{2, 4, 5}; // массив из целых чисел 2, 4, 5
    String stroka = "govno"; // строковая переменная
    String[] stroki = new String[]{"s1", "s2", "s3"}; // массив из строк s1, s2, s3
}
```

Используемые коллекции и их методы:
```java
public class Main {
    String a = "zz";
    Set<String> set = new HashSet<>(); // Множество (массив, где все элементы уникальны)

    String b = "xxx";
    Map<Integer, Boolean> map = new HashMap<>();
     // массив из пар ключ - значение (в данном случае ключи - целые числа, а значения - boolean переменные)

    void example() {
        set.add(a); // добавить в set строку a
        set.remove(a); // удалить из set строку a
        set.contains(a); // метод, который проверяет, есть ли строка а в set (если есть - вернет true)

        map.put(1, true); // добавить в мап пару 1 - true
        map.get(1); // получить из мапа значение ключа 1
    }
}
```

Используемые циклы:
```java
public class Main {
    void example() {
        for (let i = 0; i < 4; i++) { // от i = 0 до i < 4, прибавляя на каждой итерации к i 1
            a += i; // a = a + i
        }
        
        for (int b: new int[]{1, 2, 4}) { // перебираем все элементы массива 1, 2, 4 и ложим итерируемый элемент в переменную b
            a += b; // a = a + b
        }

        do { // делаем это, пока не перестанет выполняться условие напротив while:
            a++; // прибавить к а 1
        } while (a < 0); // покуда а не станет меньше 0 (ну да, шиза)
    }
}
```

### Условия

If else:
```java
public class Main {
    int a = 15;

    void example() {
        if (a < 10) {
            a = 20; // если a меньше 10 - присвоить a значение 20
        } else {
            a = 0; // если нет - присвоить a значение 0
        }
    }
}
```

Switch:
```java
public class Main {
    int a = 1; 
    
    void example() {
        switch (a) { // передаем в switch переменную a
            case 0: // если a = 0
                a = 10; // присваиваем a значение 10
                break; // выход из свитча
            case 1: // если a = 1
                a = 20; // присваиваем a значение 20
                break; // выход из свитча
            default: // если не сработало ни одно из вышестоящих условий
                a = 0; // присваиваем a значение 0
                break; // выход из свитча
        }
    }
}
```
